# 🚛 Tata Ace Vegetables Sales App

ஒரு single HTML file-ல் இயங்கும் முழுமையான காய்கறி விற்பனை கணக்கு app.  
No internet required · No installation · Works in any browser

---

## 📱 How to Use

1. `veg_sales_app.html` file-ஐ download பண்ணுங்க
2. Chrome browser-ல் open பண்ணுங்க
3. Ready! Data automatically save ஆகும்

---

## ✨ Features

### 📦 Stock Tab
- காய்கறி stock add / edit / delete
- வாங்கிய விலை, விற்கும் விலை, kg சேர்க்கலாம்
- Decimal kg support (1.5kg, 0.250kg)
- மொத்த முதலீடு, விற்பனை மதிப்பு, லாபம் summary
- Customer bill save ஆகும்போது stock automatically கழிக்கும்
- 🟢🟡🔴 Remaining stock color indicator

### 🧾 Customer Bill Tab (Front Page)
- **2 customers-க்கு ஒரே நேரத்தில் bill** பண்ணலாம்
- Stock-ல் உள்ள vegetables auto-suggestion
- kg type பண்ணா amount auto-calculate
- Decimal support (0.250kg, 1.5kg)
- Keyboard மூடாம continuously type பண்ணலாம்
- Bill save + WhatsApp share + Copy option
- Bill save ஆகும்போது stock-ல் இருந்து kg கழிக்கும்

### 💰 Customer லாபம் Tab
- Date வாரியா profit பார்க்கலாம்
- **Shift-wise profit:**
  - 🌅 Morning Shift (6AM – 2PM)
  - 🌙 Evening Shift (2PM – 10PM)
  - 📊 All Shifts (இரண்டும் தனித்தனியா)
- ஒவ்வொரு customer-க்கும் தனி profit breakdown
- Stock buy rate கழிச்சு real profit கணக்கு

### 📜 Customer History Tab
- Date & name filter
- தொகை hide/show toggle (👁 பாதுகாப்பு)
- Delete option

### 💸 செலவு & Waste Tab
- பெட்ரோல், கூலி போன்ற செலவு add/edit/delete
- Waste vegetables kg & நஷ்டம் கணக்கு
- Decimal support

### 📊 Full Summary Tab
- Stock details (kg, rate, profit per vegetable)
- நிதி சுருக்கம் (விற்பனை − செலவு − waste = நிகர லாபம்)
- Date filter
- 📄 Text Download
- 🖨️ Print / PDF Save
- 📲 WhatsApp Share

| Location | Tabs |

| Front Page (always visible) | 🧾 Customer Bill |

## 🗑️ Clear Data

ஒவ்வொரு tab-லயும் **"All Clear"** button இருக்கு:
- Stock clear
- History clear
- செலவு clear / Waste clear
- Customer 1 & 2 Bill clear

---

## 💾 Data Storage

- எல்லா data-உம் **browser localStorage**-ல் save ஆகும்
- Internet தேவையில்லை
- Same phone, same browser-ல் data இருக்கும்
- Browser clear பண்ணா data போகும் — backup-க்கு Summary download பண்ணுங்க

---

## 📲 Install as App (PWA)

1. Chrome-ல் open பண்ணுங்க
2. 3 dots menu → **"Add to Home Screen"**
3. Phone-ல் app icon வரும் — app மாதிரியே திறக்கும்!
